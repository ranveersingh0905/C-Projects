## 💻 Compiler

These programs can be compiled using:

- GCC
- Code::Blocks
- Turbo C (for educational purposes)
- Visual Studio Code (with GCC/MinGW)

Example compilation:

```bash
gcc filename.c -o output
./output
```

On Windows:

```bash
gcc filename.c -o output.exe
output.exe
```
